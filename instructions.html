General Gameplay

Diablo is an action role playing game. You will move you character with your mouse. You will have to cast spells and melee attack enemies to kill them. You use spells and attacks with youe keyboard. You will have to find these items, spells and weapons to kill the enemies. You will have to go through dungeons and kill enemies to beat this game. Each dungeon has a theme that will stick throughout the dungeon.

Classes

First class would be the warrir this is a melee class focusing on strength. This class uses mainly melee weapons to kill the enemies and get through the game. The warrior starting skill is a skill that repairs objects. The second class is the rogue. This class mainly uses ranged weapons. He uses a bow and arrow to kill the enemies. The rogues starting skill is a skill that disarms traps. The third class is the sorcerer. This class focuses on magic and spells. Whiel he is the most weak he can use the strongest spells. The starting skill for the sorcerer is a skill that recharges spells.
